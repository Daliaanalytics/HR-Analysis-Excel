# HR-Analysis-Excel

## Project Overview
An HR analytics project analyzing employee data including salaries, performance,
recruitment sources, and absences using Excel/Google Sheets.

## Tools Used
- Google Sheets / Excel
- Pivot Tables
- Charts & Dashboard

## Dataset
HR Dataset with 311 employees including salary, department, performance score,
gender, recruitment source, and absences.

## Analysis
- Average Salary by Department
- Gender Distribution
- Average Salary by Performance Score
- Recruitment Source Distribution
- Average Absences by Department

## Key Findings
- **Executive Office** has the highest average salary at **$250,000**
- **56.6%** of employees are female
- **Exceeds** performers earn the highest average salary at **$77,144**
- **Indeed** is the top recruitment source with **87 hires**
- **Sales** department has the highest average absences at **11.5 days**
